# 월간 근무표 자동화 앱

브라우저에서 `index.html`을 열어 사용하는 정적 웹앱입니다. 별도 설치나 서버가 필요하지 않습니다.

## 주요 기능

- `N`, `D`, `E`, `H`, `M1`, `M2`, `M3`, `P`, `S`, `R`, `W`, `X`, `A` 근무 모듈 드래그 앤 드롭
- 직원별 그룹, `+1`, `H/D/P` 인력풀, 이전 달 마지막 4일 데이터 입력
- 공휴일, 목표 휴무, N 간격, 연속근무 제한, 평일 가동인원 기준 설정
- 자동 생성 후 규칙 검증 및 위반 셀 표시
- `Ctrl+Z`로 직전 작업 되돌리기
- 드래그 앤 드롭 시 새 검증 문제가 생기지 않는 셀에만 배치
- `N`과 다음날 `W`는 하나의 세트로 같이 교환
- 월간 `N/W` 선배정 엔진, 금-토/토-일 `N/W` 반복 회피 및 보정
- 통계 탭의 일별 휴무/근무자 집계
- `초기배치` / `검증편집` 모드 분리
- 브라우저 저장, 인쇄, CSV 가져오기/내보내기

## 사용 순서

1. `직원` 탭에서 직원명, 그룹, 인력풀, D-4~D-1 데이터를 입력합니다.
2. 왼쪽 패널에서 기간, 공휴일, 검증 기준을 조정합니다.
3. `자동 생성`을 누릅니다.
4. `검증` 탭에서 위반 항목을 확인합니다.
5. 필요한 셀은 왼쪽 `근무 모듈`을 드래그해서 직접 수정합니다.

수동으로 수정한 셀은 다음 자동 생성 때 보호됩니다. 전체 배정을 다시 만들고 싶으면 `초기화`를 먼저 누르세요.

근무 셀을 드래그하면 같은 날짜 열에서 교환 가능한 직원 셀만 초록색으로 강조되고, 나머지는 어둡게 표시됩니다. 셀끼리 드래그하면 두 셀의 근무가 서로 교환됩니다.

`CSV 내보내기`는 기간, 검증 기준, 공휴일, 직원 설정, 근무표, 수동 수정 표시를 한 파일에 저장합니다. `CSV 가져오기`로 같은 파일을 다시 불러오면 현재 앱 상태가 해당 CSV 내용으로 복원됩니다.

`초기배치 모드`에서는 H 누락, 가동 인원 부족 같은 완성표 검증이 수기 배치를 막지 않습니다. `자동 생성`을 누르면 `검증편집 모드`로 전환되고, 이후에는 새 검증 문제가 생기는 배치가 제한됩니다.

`섞기 번호`를 비워두면 기간과 직원 목록을 기반으로 자동 시드를 사용합니다. 같은 달에도 다른 패턴을 보고 싶으면 `1`, `2`, `3`처럼 값을 바꾼 뒤 다시 자동 생성하세요. 같은 섞기 번호는 같은 조건에서 같은 결과를 재현합니다.
